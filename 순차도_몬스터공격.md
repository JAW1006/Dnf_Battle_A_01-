```mermaid
sequenceDiagram
    autonumber
    actor 사용자 as 플레이어
    participant UI as Attack_Monster_UI
    participant Control as 전투
    participant PlayerClass as 플레이어
    participant Char as 캐릭터(전사/마법사)
    
    사용자->>UI: 몬스터 공격 요청 (플레이어id)
    activate UI
    
    UI->>Control: 몬스터공격(플레이어id: String) : void
    activate Control
    
    %% <<include>> 관계인 플레이어체크 수행
    Control->>PlayerClass: 플레이어체크(플레이어id: String) : boolean
    activate PlayerClass
    PlayerClass-->>Control: boolean (true 반환 시 진행)
    deactivate PlayerClass
    
    %% 다형성을 이용한 스킬 발동 (전사 or 마법사)
    Control->>Char: 스킬발동() : float
    activate Char
    Char-->>Control: float (최종 데미지 수치 반환)
    deactivate Char
    
    %% 데미지 수치에 따른 등급 처리 로직 (컨트롤러 내부 로직)
    alt 데미지 >= 200
        Note over Control: S급 공격 판정
    else 데미지 >= 100
        Note over Control: A급 공격 판정
    else 데미지 < 100
        Note over Control: B급 공격 판정
    end
    
    Control-->>UI: void (공격 완료 및 결과 전달)
    deactivate Control
    
    UI-->>사용자: 공격 스킬 및 데미지, 등급 결과 출력
    deactivate UI
