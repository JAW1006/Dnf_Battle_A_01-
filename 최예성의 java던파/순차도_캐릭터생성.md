```mermaid

sequenceDiagram
    autonumber
    actor User as 플레이어(사용자)
    participant UI as Create_Character_UI
    participant Control as 전투
    participant PlayerClass as 플레이어(클래스)
    
    User->>UI: 정보 입력 (플레이어id, 캐릭터명, 직업, 레벨)
    activate UI
    
    UI->>Control: 캐릭터생성(플레이어id: String, 캐릭터명: String, 직업: String, 레벨: int) : void
    activate Control
    
    %% <<include>> 관계인 플레이어체크 수행
    Control->>PlayerClass: 플레이어체크(플레이어id: String) : boolean
    activate PlayerClass
    PlayerClass-->>Control: boolean (true 반환 시 진행)
    deactivate PlayerClass
    
    %% 상단 선언 없이 생성 시점에 ID와 이름을 함께 정의
    alt 직업 == "전사"
        create participant Warrior as 전사
        Control->>Warrior: <<create>> 생성(캐릭터명, 레벨, HP=레벨*100, 공격력=레벨*15)
        activate Warrior
        Warrior-->>Control: void
        deactivate Warrior
    else 직업 == "마법사"
        create participant Mage as 마법사
        Control->>Mage: <<create>> 생성(캐릭터명, 레벨, HP=레벨*60, 공격력=레벨*25)
        activate Mage
        Mage-->>Control: void
        deactivate Mage
    end
    
    Control-->>UI: void (생성 완료)
    deactivate Control
    
    UI-->>User: 캐릭터 생성 완료 안내
    deactivate UI
```
