```mermaid

sequenceDiagram
    autonumber
    actor 사용자 as 플레이어
    participant UI as Join_Guild_UI
    participant Control as 전투
    participant PlayerClass as 플레이어(클래스)
    participant Guild as 길드(대상길드)
    
    사용자->>UI: 길드 가입 신청 (플레이어id)
    activate UI
    
    %% 사전에 UI(application 영역)에 존재하는 길드 객체를 함께 전달
    UI->>Control: 길드가입(플레이어id, 대상길드) : void
    activate Control
    
    %% <<include>> 플레이어 체크 수행
    Control->>PlayerClass: 플레이어체크(플레이어id) : boolean
    activate PlayerClass
    PlayerClass-->>Control: true (반환)
    deactivate PlayerClass
    
    %% 대상 길드에 현재 캐릭터의 주소값을 넘겨주며 가입 요청 (Aggregation)
    Control->>Guild: 캐릭터가입(현재캐릭터) : boolean
    activate Guild
    
    alt 길드 인원(5) 미만
        Note over Guild: 캐릭터리스트에 현재캐릭터<br/>참조 추가 (Aggregation)
        Guild-->>Control: true
    else 길드 정원 초과
        Note over Guild: 최대인원 초과로 가입 거부
        Guild-->>Control: false
    end
    deactivate Guild
    
    Control-->>UI: void
    deactivate Control
    
    %% UI 단에서 가입 전후 인원수 비교로 결과 판정
    Note over UI: UI에서 가입 전후 길드<br/>인원수 비교를 통해 성공 여부 판정
    UI-->>사용자: 길드 가입 결과 및 길드원 현황 출력
    deactivate UI
