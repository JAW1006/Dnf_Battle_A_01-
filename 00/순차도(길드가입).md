```mermaid
sequenceDiagram
    autonumber
    actor User as 플레이어
    participant UI as Join_Guild_UI
    participant System as 전투
    participant P as 플레이어
    participant Guild as 길드
    
    User->>UI: 정보 입력(플레이어id, 길드명)
    activate UI
    UI->>System: 길드가입(플레이어id, 길드명)
    activate System
    
    System->>P: 플레이어체크(플레이어id)
    activate P
    P-->>System: boolean
    deactivate P
    
    alt 검증 통과
        System->>Guild: 캐릭터가입(현재 캐릭터 객체)
        activate Guild
        
        alt 길드 인원 < 5
            Guild->>Guild: 캐릭터리스트에 캐릭터 추가 (참조 연결)
            Guild-->>System: true (가입 성공)
            System-->>UI: 길드 가입 완료 완료 안내
        else 길드 정원 초과 (>= 5)
            Guild-->>System: false (가입 실패)
            System-->>UI: 에러 메시지 (길드 정원이 가득 찼습니다)
        end
        deactivate Guild
    else 검증 실패
        System-->>UI: 에러 메시지 (플레이어 체크 실패)
    end
    deactivate System
    deactivate UI
