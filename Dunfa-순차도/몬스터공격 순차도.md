```mermaid
sequenceDiagram
    actor 사용자 as 플레이어
    participant UI as Attack_Monster_UI
    participant 전투
    participant 플레이어클래스 as 플레이어
    participant 캐릭터
    participant 전사
    participant 마법사

    사용자->>UI: 몬스터공격(플레이어id)
    UI->>전투: 몬스터공격(플레이어id)
    
    전투->>플레이어클래스: 플레이어체크(플레이어id)
    플레이어클래스-->>전투: boolean
    
    alt 플레이어id == "hero" AND 현재캐릭터 != null
        전투->>캐릭터: 스킬발동()
        
        alt 현재캐릭터 == 전사
            캐릭터->>전사: 스킬발동()
            전사-->>캐릭터: 데미지: float (공격력x1.5)
        else 현재캐릭터 == 마법사
            캐릭터->>마법사: 스킬발동()
            마법사-->>캐릭터: 데미지: float (공격력x2.0)
        end
        
        캐릭터-->>전투: 데미지 반환 (float)
        전투->>전투: 결과_스킬명, 결과_데미지 변수에 저장

        alt 데미지 >= 200
            전투->>전투: 결과_등급 = "S급 공격" 저장
        else 데미지 >= 100
            전투->>전투: 결과_등급 = "A급 공격" 저장
        else 데미지 < 100
            전투->>전투: 결과_등급 = "B급 공격" 저장
        end
        
        전투->>전투: 공격성공여부 = true 상태 저장
    else 권한 불일치 혹은 캐릭터 없음
        전투->>전투: 에러 메시지 저장
    end

    전투-->>UI: void
    
    alt 공격 성공 시
        UI-->>사용자: 발동스킬, 데미지, 등급 화면 출력
    else 공격 실패 시
        UI-->>사용자: 에러 메시지 화면 출력
    end
